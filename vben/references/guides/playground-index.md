# Playground 示例索引

Playground 是学习 vben-admin 最好的资源。

## 示例目录

```
playground/src/views/examples/
├── form/              # 表单示例
├── modal/             # 弹窗/抽屉示例
├── page/              # 页面示例
├── table/             # 表格示例
└── vxe-table/         # VxeGrid 表格示例
```

## 表单示例

| 示例 | 路径 | 说明 |
|------|------|------|
| 基础表单 | `examples/form/basic.vue` | 表单基础用法 |
| 查询表单 | `examples/form/query.vue` | 表格上方查询表单 |
| 表单联动 | `examples/form/linkage.vue` | 字段联动 |
| 表单校验 | `examples/form/rules.vue` | 校验规则 |
